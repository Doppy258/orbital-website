FAQ

Your questions,
answered with clarity

Do I need to leave my laptop on? +
No. All execution happens in cloud sandboxes. Your laptop can be off, closed, or at home. Orbital spins up an ephemeral container for every task.
Which repos can Orbital access? +
Any GitHub repo you connect. Orbital reads your code, branches, and history. It can also open PRs and commit directly with your approval.
Which AI models does it use? +
Orbital supports Claude, GPT-4, DeepSeek, and Code Llama. You can configure which model runs your agent tasks in settings.
Is my code secure? +
Yes. Every task runs in a private, isolated container that's destroyed after the task completes. We never store your code. All connections are encrypted end-to-end.
